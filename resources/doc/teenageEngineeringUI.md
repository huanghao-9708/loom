A high-end desktop music player interface design inspired by Teenage Engineering, industrial design, and professional audio equipment UI.

The UI should look like a precision-built music workstation rather than a software app.

Design style:
- ultra minimal but highly structured
- monochrome base (white / light gray / soft black)
- strict grid system with precise alignment
- no gradients, no glassmorphism, no decoration
- typography-driven layout with strong hierarchy
- technical, engineering-grade aesthetic

Layout:
- left sidebar navigation (library, albums, artists, playlists, sources)
- central main panel showing a dense track list in table format
- right panel showing now playing, album art, lyrics, and queue
- bottom fixed playback control bar like a hardware audio deck

UI behavior feel:
- resembles a professional DJ or audio engineer workstation
- resembles hardware UI from synthesizers and studio gear
- every element feels functional, not decorative

Typography:
- modern sans serif (Inter, Helvetica Neue, or custom industrial font)
- uppercase labels for system sections
- tight spacing but highly readable
- numeric metadata clearly aligned (time, format, bitrate)

Color system:
- base: warm light gray background
- surface: white panels
- text: near-black and mid-gray hierarchy
- accent: single muted amber or orange used only for "playing state"
- active track highlighted with thin left indicator bar

Key interaction elements:
- active track row highlighted with strong contrast inversion
- waveform visualization embedded in track row
- subtle separators instead of heavy borders
- iconography is thin-line, technical, minimal

Bottom player:
- resembles physical audio hardware deck
- includes play/pause, skip, shuffle, repeat
- progress bar looks like precision measurement scale
- volume control resembles analog knob or slider

Overall mood:
- Teenage Engineering OP-1 meets Apple Music Library meets industrial UI design
- extremely clean, precise, engineered, quiet luxury
- feels like a tool for professionals, not a content app

Render as a high-resolution UI mockup, desktop 16:9, ultra sharp, realistic product design presentation, soft studio lighting, slight shadow depth but no heavy effects
